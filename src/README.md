Weekly Planner - Backend API

Esta es la API que alimenta a la aplicación del Weekly Planner. Maneja la lógica de negocio, la seguridad, la base de datos y el envío de correos electrónicos.


Tecnologías

Runtime: Node.js
Framework: Express.js
Base de Datos: MongoDB (Mongoose)
Seguridad: JSON Web Tokens (JWT) & Bcrypt
Mails: Nodemailer (Gmail SMTP)
Validaciones: Middlewares personalizados
